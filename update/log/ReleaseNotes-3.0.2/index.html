XRCC 3.0.2 Release

January 22, 2025 · 4 min read

New

  • Added open file location on project thumbnail in PC homepage
  • Added more information in Project Information Panel
  • Added model animation preview in inspector
  • Added mode selection on open project in XR mode

Updated

  • Updated Reporter Panel to require name and email
  • Improved UX on Reporter Panel to support multi-select to update files
  • Updated project name synchronizing to file name
  • Improved performance on Item Menu
  • Improved performance on update physics collider in Text Object
  • Improved UX on light preview range of Light Object
  • Updated entity position and rotation in play mode to use Edit mode data in PC play mode
  • Improved UX on Action Selection Panel
  • Improved UX on loading package in XR mode
  • Updated keyboard control for move up and down in XR mode
  • Updated implementation for Empty Object with isWall attribute

Bug fixes

  • Fixed an issue where some settings on UI are not stored in setting data
  • Fixed an issue where color picker panel is not working as expected
  • Fixed an issue where undo/redo action are not working as expected after entering play mode
  • Hotfixed an issue where undo/redo action are not working as expected when the operation is dealing with large amount of items
  • Fixed some typos in behaviours block names
  • Fixed an issue where text overlay is not working and overlay on top of images
  • Fixed an issue where items are unable to parent to Light Object
  • Fixed an issue where teleporter tool is not working as expected
  • Fixed an issue where saving is not working in scene panel
  • Fixed an issue where auto-save failed to function
  • Fixed an issue where fine-tuning of input values in popup panel is not working when the value is default value
  • Fixed an issue where model asset not requesting recenter when import model from file and Sketchfab
  • Fixed an issue where Skybox image could not update when pressing the remove button
  • Fixed an issue where change scene asset failed to update the scene thumbnail
  • Fixed an issue where scene thumbnail in inspector is not updated as expected after changing scene asset
  • Fixed an issue where scene name in inspector not updated as expected when using scene from package
  • Fixed an issue where scene thumbnail could not be read from package
  • Fixed an issue where ccdata file can be selected when import asset
  • Fixed an issue where model rotation and scale from package are not assigned as expected
  • Fixed an issue where parenting sometimes not work when trying to link a child item to parent item already have two levels of children item
  • Fixed an issue where calling Fade Behaviour in play mode will affect input and UI when going back to edit mode
  • Fixed an issue where behaviour layer panel not refresh on new layer created by other panel
  • Fixed an issue where Action Start Behaviour can be selected and duplicated in behaviour panel
  • Hotfixed an issue where Set Model behaviour wrongly set color to white
  • Fixed an issue where Play Sound Behaviour failure will stop all subsequent behaviours when no audio asset is assigned
  • Fixed an issue where Start Contact Behaviour is not working with entity
  • Fixed an issue where behaviour variables lost reference when the variable is renamed
  • Fixed an issue where change material color is not working as expected in behaviour
  • Fixed an issue where player tool does not change to wand when opening ring menu in XR mode
  • Fixed an issue where fade is not working as expected in XR mode
  • Fixed an issue where gravity setting is not working as expected in XR mode
  • Fixed an issue where move the player up & down may fall through the ground in XR mode
  • Fixed an issue where asset sometimes cannot be spawned in XR mode
  • Fixed an issue where item display not working as expected after updating Hide In Edit and Hide In Play attribute
  • Fixed an issue where Start Contact Behaviour triggered twice when making contact to user
