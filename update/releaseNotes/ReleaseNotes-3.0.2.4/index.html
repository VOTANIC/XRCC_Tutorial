XRCC 3.0.2.4 Release

February 20, 2025 · One min read

Updated

  • Updated file browser allowing import of multiple attachments in Reporter Panel
  • Updated item and user to ignore gravity when they are moving through the "Set Position" behaviour
  • Updated Ground Object to diasble collider when "Hide In Play" is enabled

Fixed

  • Fixed an issue where notification show up unexpectedly
  • Fixed an issue where network mode may not be connected after disconnection
  • Fixed an issue where color panel may not work properly
  • Fixed an issue where undo/redo operation not working as expected in model asset
  • Fixed an issue where Waypoint Object not hiding after enable "Hide In Play"
  • Fixed an issue where Play audio override the last playing audio
  • Fixed an issue where children under entity not able to select in get/set variable
  • Fixed an issue where reset not working with correct default value in inspector of entity
  • Fixed an issue where modify item's "Hide In Play" and "Lock" option cannot be saved to project
  • Fixed an issue where default color value of OptionBox Object is wrong
  • Fixed an issue where behaviour flow is stopped if invoking "Call Action" behaviour without calling an action
  • Fixed an issue where Entity cannot invoke Stop Contact Behaviour
