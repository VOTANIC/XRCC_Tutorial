Application Level

  • Online Text and Video Tutorial
  • Redesigned user interfaces for user experiences

Project Level

  • Favorite feature for project
  • Metadata for project, e.g. project thumbnail, description

  • Asset
    • Redesigned unified asset management panel
    • Asset tag feature
    • More default assets
    • Allow model to use multiple animations
    • Character and motion upgrade
