Main Menu

Main Menu located at the top of the frame which mainly separated into five categories: File, Project, Edit, View and Help.

FileProjectEditViewHelp

TabButtonDescription
File
New...
Create a New project.
Open...
Open an existing project.
Save Project
Save the currect project.
Save As...
Save project at choose location.
Export Image
Capture and export an image.
Import Assets
Import assets from user computer.
Add New Scene
Add a new scene to currect project.
Exit
Return to home page.
Project
Asset Menu
Show assets panel.
Project Information
Show project information panel.
Manage Packages
Show import package panel
Edit
Undo
Undo the last action.
Redo
Redo the last action.
Copy
Copy the selected object(s).
Paste
Paste the last copied object(s).
Duplicate
Duplicate the selected object(s).
Delete
Delete the selected object(s).
Link
Link object(s) to set it as a child of another object.
Unlink
Unlink the linked objects.
View
Statistics
Show the statistics.
Inspector
Show the Inspector panel.
Object Pop-up Panel
Show the object pop-up panel.
Control Hints
show control hints.
Normal Screen Mode
Switch to normal screen mode
Full Screen Mode
Switch to full screen mode.
Help
Tutorials...
Open the url of tutorials.
About XRCC
Show introduction of XRCC.
Shortcut List
Show the shortcut panel.
My Settings
Show setting panel.
Feature Requests/Bug Reports
Show the report panel.
Sign in/out SketchFab
Sign in/out SketchFab account if user has logged in.
