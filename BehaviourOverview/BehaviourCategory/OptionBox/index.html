Option Box

note

Option Box Blocks are available for Option Box object type only.

Block Name
Block
Description
Show Option Box
Show the option box.
  • Show: Boolean value specifying visibility of the option box.
Set Content Text
Set text in the content of the option box.
  • Text: String value specifying the content text of the option box.
Show Content Image
Show the option box image.
  • Show: Boolean value specifying visibility of the option box image.
Set Title Text
Set text in the title of the option box.
  • Text: String value specifying title text of the option box.
Set Content Image
Set image in the content of the option box.
  • Image: Image asset specifying content image of the option box.
Use Options
Utilizes the option.
  • Use: Boolean value specifying if the option is used.
