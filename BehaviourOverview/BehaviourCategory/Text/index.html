Text

note

Text Blocks are available for Text object type only.

Block NameBlockDescription
Set Text

Set the text to display.

  • Text: Text value specifying the text to display.
Set Font Size

Set the font size of the text.

  • Size: Integer value specifying font size of the text.
Set Font Color

Set the font color of the text.

  • Color: Color value specifying font color of the text.
Set Background Color

Set the background color of the text.

  • Color: Color value specifying background color of the text.
