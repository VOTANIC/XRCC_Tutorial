Effect

note

Effect Blocks are available for all object types except Scene.

Block NameBlockDescription
Show

Set the visibility of the object.

  • Enable: Boolean value specifying the visibility of the object.
Outline

Outline the object with specified color.

  • Color: Boolean value specifying the color of the outline.
  • Enable: Boolean value specifying the visibility of the outline.
Overlay

Outline the object with specified color.

  • Color: Boolean value specifying the color of the overlay.
  • Enable: Boolean value specifying the visibility of the overlay.
Blinker

Outline the object with specified color.

  • Color: Boolean value specifying the color of the blinker.
  • Enable: Boolean value specifying the visibility of the blinker.
