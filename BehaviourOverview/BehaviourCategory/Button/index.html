Button

Block NameBlockDescription
Get Button Property

Get different button property.

  • Text: The Text value specifying the text displayed on the button.
  • Font Color: The Color value specifying the color of the text displayed on the button.
  • Font Size: The Integer value specifying the size of the text displayed on the button.
Set Button Property

Set different button property.

  • Text: The Text value specifying the text displayed on the button.
  • Font Color: The Color value specifying the color of the text displayed on the button.
  • Font Size: The Integer value specifying the size of the text displayed on the button.
Set Normal Color

Set the normal color of the button.

  • Normal Color: The Color to display the button normally.
Set Hover Color

Set the color to render the button as when the player hovers over it.

  • Hover Color: The Color to display the button when it is hovered over.
Set Select Color

Set the color to render the button as when it is selected.

  • Select Color: The Color to display the button when it is selected.
