Light

note

Light Blocks are available for Light object type only.

Block NameBlockDescription
Get Light Property

Get different light property.

  • Intensity: Number value specifying the intensity of the light.
  • Color: Color value specifying the color of the light.
  • Range: Number value specifying the range of the light.
  • Cast Shadow: Boolean value specifying the visibility of the cast shadow.
  • Angle: Number value specifying the angular degree of light ray with respect to the object's xyz axis.
Set Light Property

Set different light property.

  • Intensity: Number value specifying the intensity of the light.
  • Color: Color value specifying the color of the light.
  • Range: Number value specifying the range of the light.
  • Cast Shadow: Boolean value specifying the visibility of the cast shadow.
  • Angle: Number value specifying the angular degree of light ray with respect to the object's xyz axis.
