Light

note

Light Blocks are available for Light object type only.

Block NameBlockDescription
Set Angle

Set the angle of the light.

  1. Set Angle: Number value specifying the angular degree of light ray with respect to the object's xyz axis.
Set Color

Set the color of the light.

  1. Color: Color value specifying color of the light.
Set Intensity

Set the intensity of the light.

  1. Intensity: Number value of light intensity
Set Range

Set the range of the light.

  1. Range: Number value of wide range of light.
Set Cast Shadow

Set the visibility of cast shadow.

  1. Enable: Boolean value specifying the visibility of the cast shadow.
