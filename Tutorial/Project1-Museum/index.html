XRCC Tutorial - Museum

Overview

You will build an interactive VR museum using XRCC in this tutorial. The techniques you'll learn in the tutorial are fundamental to building any XR experiences using XRCC, going through this tutorial will help you gain a basic understanding of the scene building process. We recommend that you start with this tutorial if it's the first time you are using the XRCC.

You will learn the following techniques in this tutorial:

Tutorial Projectâ€‹

The project used for this tutorial comes in two formats: a starter project for use in this step-by-step tutorial, and a completed project for your reference after you've completed the tutorial.

Project files for the museum tutorial can be found at:

For this tutorial, the starter project will be used to guide you build the VR museum in a step-by-step manner. All the necessary assets required for the tutorial are already included in the project, so just open the starter project in XRCC and you are ready to go with the tutorial.

Build the Scene and Navigation Area

Build the Sceneâ€‹

  1. From the Toolbar, press "Spawn New Object" to launch the New Item Panel, click the "Imported Assets" tab and select the "art gallery.fbx" 3D model to add it to the scene.

  1. Open the "Configure Tool" and deselect "Interactable" for the art gallery object. As the art gallery is just a background environment in the scene, deselecting Interactable prevents the VR wand from interacting with the object.

Lay Out the Navigation Areaâ€‹

  1. Open the "New Item Panel" and from the "Specialized Objects" tab, select and add a "Ground" object to the scene. The ground object allows user to stand on and move / teleport on the surface.
  1. Use the transform tools (move / rotate / scale) to adjust the size and position of the ground object, then open the "Configure Tool" and turn off "Interactable" for the ground object as well.
  1. Open the "New Item Panel" and from the "Specialized Objects" tab, select and add an "Empty Object" to the scene. We will make use of the empty objects and configure them to act as virtual walls to prevent the user from moving outside the boundaries of the art gallery.
  1. Use the transform tools (move / rotate / scale) to adjust the size and position of the empty object so that it completely surrounds the wall surface of the art gallery's entrance behind the user.

  1. Open the "Configure Tool" and deselect "Interactable" for the empty object. Then select "Enable Physics" and "Static" under physics setting in the configure tool. Setting the empty object as a static physics object ensures the empty object will remain at same place when user collides with it.

  2. With the empty object still selected, Copy (Ctrl + C) and Paste (Ctrl + V) or Duplicate (Ctrl + D) the configured empty object static wall and use the transform tools to set up the remaining walls of the art gallery.

  1. You have finished configuring the navigation area of the VR museum. Press the "Play" button and test walking around in the scene. You should be able to stand on the ground and move using the WASD key, and the walls should stop you when you walk towards them.
  1. Press "Esc" to exit play mode after you've done testing your scene.

Create the Museum Guide

Add a Character as Museum Guideâ€‹

  1. Open the "New Item Panel" and from the "Default Assets" tab, select and add a Female Adult character model to the scene.
  1. Use the transform tools (move / rotate / scale) to reposition the character to in front of and facing the user.

  1. Open the "New Item Panel" and from the "Specialized Objects" tab, select and add a "Text Object" to the scene.
  1. Use the transform tools (move / rotate / scale) to reposition the text object next to the female character. Then open the "Configure Tool" to set up the text content, font size, and colour of the text object.

Make the Character Move and Speakâ€‹

Set Character to Speak and Move when Scene Startsâ€‹

  1. Select the female character object and click on the "Behaviour" tab to switch to its behaviour panel.
  1. In the "General Event" category, find and drag the "Start" event trigger block onto the behaviour canvas. Then switch to the "Character" category, find and drag both the "Play Voice" and "Play Motion" behaviour blocks onto the behaviour canvas. Connect the "Start" block's out flow node with the "Play Voice" block's in flow node, and "Play Voice" block's out flow node with "Play Motion" block's in flow node. This behaviour flow defines a behaviour sequence for the character model that begins at the "Start" of the scene, it continues with the character playing a voice clip followed by a motion.
  1. On the "Play Voice" block, click the selector under the "Audio" option to open the Selection panel and select the "Introduction.mp3" audio clip from your asset library. For the "Play Motion" block, click the selector under the "Motion" option to open the Selection panel and select the "Hello" motion from the asset library.

Set Character to Move and Speak when Clickedâ€‹

  1. On the female character's behaviour canvas, select both the "Play Voice" and "Play Motion" blocks on the canvas, Copy (Ctrl + C) and Paste "" (Ctrl + V) or Duplicate (Ctrl + D) the block and move them to a lower position on the canvas.
  1. Switch to the "Item Event" category, find and drag the "Click" event trigger block onto the behaviour canvas. Connect the "Click" block's out flow node with the "Play Motion" block's in flow node, and "Play Motion" block's out flow node with "Play Voice" block's in flow node. This behaviour flow defines a behaviour sequence for the character model that begins when user "Click" the character using the wand, it continues with the character playing a motion followed by a voice clip.

  1. You have now finished setting up the behaviours for the character model. Press the "Play" button () and watch the character greet you when the scene starts, and when clicked at.

  1. Press "Esc" to exit play mode after you've done testing your scene.

Add the 2D Exhibits

Add 2D Imageâ€‹

  1. Open the "New Item Panel" and from the "Specialized Objects" tab, select and add a "2D Image Object" to the scene. This time, use the spawn object button at the centre to spawn the 2D image object to the centre of the screen.
  1. Open the "Configure Tool", click the selector next to the "Image Asset" option to open the Selection panel and select an image in your asset library to display the image in scene.
  1. Use the transform tools (move / rotate / scale) to reposition the 2D image object to the wall on the front row.

  1. Make a few copies of the 2D Image objects and change their Image Assets, then place the objects to the empty areas on the front wall.

Add 2D Videoâ€‹

  1. Open the "New Item Panel" and from the "Specialized Objects" tab, select and add a "2D Video Object" to the scene.
  1. Open the "Configure Tool", click the selector next to the "Video Asset" option to open the Selection panel and select a video in your asset library to play the video in scene.

  1. Use the transform tools (move / rotate / scale) to reposition the 2D video object to the wall on the middle row.

  1. Duplicate the 2D Video object and place it to the other display wall in the middle row, use another video in your asset library for this video asset.

Add PDF Documentâ€‹

  1. Open the "New Item Panel" and from the "Specialized Objects" tab, select and add a "Document Object" to the scene.
  1. Open the "Configure Tool", click the selector next to the "Document Asset" option to open the Selection panel and select a pdf document in your asset library to show the document in scene.
  1. Open the "New Item Panel" and from the "Imported Assets" tab, select and add the "pull up banners.glb" 3D model to the scene.
  1. Use the transform tools (move / rotate / scale) to place the document object in front of the pull up banners 3D model object. Then select both objects and position them at the left side of the back row wall.

  1. Select the document object and open its behaviour panel. Under "Groups", add a new behaviour group named "PreviousPage" and toggle the "Action" button to make it an action that can be called by other objects. An "Action Start" event trigger block will appear on the canvas.
  1. Switch to behaviour Library, under the "Document" category drag the "Set Previous Page" block and append it behind the "Action Start" block. Then connect the "Action Start" block and the "Set Previous Page" block. When this action is called, the document display will be set to the previous page.

  1. Repeat the same procedure to create the "NextPage" action.

Add Buttonsâ€‹

  1. Open the "New Item Panel" and from the "Specialized Objects" tab, select and add a "Button Object" to the scene. Use the transform tools to position the button object on the wall next to the document object.
  1. Open the "Configure Tool" and set up the content and style of the text object. Optionally, you can add some sound effects when user interacts with the button.

  1. Click the selector next to the "Action" option to open the selection panel to set up the action call. On the selection panel, select the document object from the item list on the left side, then choose the "PreviousPage" action on the right and select the action. This will set up the "Previous" button object to call the "PreviousPage" action when the button is clicked.

  1. Repeat the above process to add and configure the "Next" button to the scene.

  2. Duplicate the document and button objects and set them up on the right-side wall.

Add Slideshowâ€‹

  1. Open the "New Item Panel" and from the "Specialized Objects" tab, select and add a "2D Slideshow Object" to the scene. A slideshow object allows you to display up to 10 images in the form of a playlist, and configure the duration / transition effects of each image individually.

  1. Use the transform tools (move / rotate / scale) to place the 2D slideshow object on the left wall of the art gallery.

  2. Open the "Configure Tool" and set up the first slide of the 2D slideshow object by clicking the selector next to the "Slide Image" option, select an image from your asset library to add the image to slideshow.

  1. Click the "+" button on the right-hand side of the "Slide" option to add a new image to the slide show. Use the ">" button to change to the newly added image slide and select another image to use for that image slide. Optionally, you can configure the duration, caption text, and transition effects for the slide.

  1. Repeat the process to add more images to the slide show.

  2. Press the "Play" button and test your scene, then use the "Esc" button to return to scene editing.

Add the 3D Exhibits

Add 3D Modelâ€‹

  1. Open the "New Item Panel" and from the "Imported Assets" tab, select and add the "display cabinet.fbx" 3D model to the scene.
  1. Use the transform tools (move / rotate / scale) to place the display cabinet object to the front of of the art gallery near the front wall.
  1. Repeat the same procedure to add the food vessel 3D model object to the scene and place it inside the display cabinet.

  1. Select the food vessel object and open its behaviour panel. Add a new action called "Animation" under the behaviour "Groups". Switch to behaviour library and select the "3D Model" category, drag the "Play Animation" behaviour block and append it behind the "Action Start" block in the canvas. When the "Animation" action is called on, the food vessel object will play the animation that's embedded in the 3D model.

Invoke the 3D Model Animationâ€‹

  1. Add a new "Button Object" to the scene. In the "Configure Tool", change the text of the button into "Animate". Then set up the action call on the "Animation" action created in the last step.
  1. Press the "Play" button and test your scene, the food vessel will play an animation to open its lid when the "Animate" button is clicked. Use the "Esc" button to return to scene editing after your testing.
  1. Duplicate the display cabinet, food vessel, and button and place them to the right-side of the art gallery. Select the duplicated food vessel 3D model and open the "Configure Tool", click the selector next to the "Model Asset" option and choose the "mask.fbx" in the selection panel. Use the transform tools to adjust the position of the mask 3D model. The action set up on the duplicated button object will automatically update to connect to the mask 3D model's "Animate" option.

Create the Spotlight and Control Panel

Add Spotlightâ€‹

  1. Open the "New Item Panel" and from the "Imported Assets" tab, select and add the "ceiling Spotlight.fbx" 3D model to the scene.
  1. Use the transform tools (move / rotate / scale) to place the spotlight object at the ceiling facing the display cabinet.
  1. Open the "New Item Panel" and from the "Specialized Objects" tab, select and add a "Light Object" to the scene.
  1. Use the transform tools (move / rotate / scale) to place the light object inside the spotlight 3D model object.
  1. Open the "Configure Tool" and set the light type to "Spot", increase the Light Intensity, Light Angle values and set the Light Color to white.

Add Light Control Panel using Option Box Objectâ€‹

  1. Open the "New Item Panel" and from the "Specialized Objects" tab, select and add a "Option Box Object" to the scene.
  1. Use the transform tools (move / rotate / scale) to place the option box to the centre of the front wall between the two display cabinet objects. Open the "Configure Tool" and set up the details of the option box object. Under "Options" section, set the "Total Options" to "3" to create 3 buttons on the option box, and click the arrow buttons next to the "Option" to switch between different option details. We added 3 options to this option box object, each will be set up to call an action to change the light color of the light object.

  1. Select the light object and open its behaviour panel. Create 3 new actions and name them "white", "yellow", and "blue".
  1. Select the "yellow" action, then switch to the behaviour library, drag and drop a "Color" block under the "Math" category to the behaviour canvas under the "Action Start" block. Set the "B" (blue) value of the Color block to 144, the color preview under the "Output" node should show a yellow color.
  1. Change to the "Light" category then drag and drop a "Set Color" block to the canvas. Connect the "Action Start" block's out flow node to the "Set Color" block's in flow node, then connect the "Color" block's "Output" node to the "Set Color" block's "Color" input node.

  1. Repeat the same procedure to set up the "white" and "blue" actions.

  2. Select the option box object and open its "Configure Tool". For the first option "White", select the light object's "white" action for its "Action" under "Option Detail". Switch to the "Yellow" and "Blue" options and repeat the procedure to set up their corresponding actions.

  1. Press the "Play" button and test your scene, the spotlight will change color when you select different options on the option box. Use the "Esc" button to return to scene editing after your testing.

  2. Duplicate the spotlight and light objects, then position the duplicated set to the ceiling pointing at the display cabinet on the right-side.

  3. Go back to select the light object on the left-side, open its behaviour panel and select the "white" action you've configured earlier. Switch to behaviour library, drag and drop the "Call Action" block under the "Flow Control" category to the canvas, and append it behind the "Set Color" block. Select the duplicated light object's "white" action under "Call Action" block's "Target Action". This call action will change the light color of the duplicated light object, meaning that selecting the option box's light color options will change light color on both lights simultaneously.

  1. Repeat the same procedure for "yellow" and "blue" actions.

  2. Press the "Play" button and test your scene again, you can see both lights changing color simultaneously when the corresponding option box button is selected.

Add Point of Interest

  1. Open the "New Item Panel" and from the "Specialized Objects" tab, select and add a "Point of Interest Object" to the scene.

  2. Use the transform tools (move / rotate / scale) to place the point of interest object in front of and facing the slideshow object. The teleport tool will automatically snap to the point of interest object in XR mode. When clicked on it with the teleport tool, the user will be transitioned to the exact position and orientation defined by the point of interest object.

  1. Deselect any objects in the scene and open the "Configure Tool" to configure the scene settings. Select "Auto" for the "Default Tool". This sets the Auto tool as the default interaction tool in XR mode. When the auto tool is pointing upright, it displays a VR wand that can be used to select and click on interactable objects; when the auto tool is pointing to the ground object or a point of interest object, it changes into a teleport tool that transitions the user to the position indicated by the teleport ring. The auto tool can only be tested using XRCC's XR mode.
