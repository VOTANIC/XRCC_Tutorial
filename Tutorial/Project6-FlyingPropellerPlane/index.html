XRCC Tutorial - Microbit Paris Flight

Overview​

In this tutorial, we'll guide you through how to make a small tutorial scene with a plane that physically rotates with your microcontroller board in XRCC.

Before you begin...

To complete this project, you will need:

  • A Microbit / ESP32 microcontroller
  • Or any microcontroller with bluetooth / serial communication features

Techniques Covered​

Technique
Specific Techniques Covered
Using User Behaviour
  • Using microcontroller nodes to set up microcontroller features in XRCC
  • Using microcontroller nodes to send or receive data from microcontroller in XRCC
Microcontroller Setup & Connection
  • Using XRCC connection panel to connect your microcontroller to XRCC
  • Using MakeCode platform to write code in your microcontroller, allowing it to send data

Set up the Propeller Plane​

Let's start with adding the propeller plane into the scene.

Add the Plane Mesh​

  1. Open the Asset Panel and select the propeller plane mesh at 3D Model > air_propeller_plane, then click Add Asset.

  1. Adjust the plane mesh's position so that it sits between the player and the Eiffel Tower. In the picture below, we have set the plane's position to (0.25, 0, 2), rotation to (0, 0, 0), scale to (1, 1, 1).
Tips

Any position works fine, as long as the plane remains within the player's line of sight

Set up XRCC-to-Microcontroller Communication​

Connect Microcontroller to XRCC​

  1. Connect your microcontroller to a power source (either your computer or an external battery pack).

  2. Go to Behavior tab and open up the Microcontroller Setup panel, then click Add Device.

  1. You should now see the Add Microcontroller Connection panel. Enter a unique device name in the text box (in the example below, we used demo_device). Depending on your microcontroller's hardware configuration, navigate to either the Microbit, Bluetooth, or Serial tab and click Discover to search for nearby devices. Finally, select your microcontroller from the list and click Connect to establish a connection.

  1. Look for a green checkmark that indicates your device has successfully connected to XRCC. Once you see the checkmark, add the device by clicking Add Microcontroller Connection.

Checkpoint

Check that your device is connected to XRCC. It should appear on the Connected Microcontroller list with a green checkmark.

Troubleshooting

If a connection failure has occured, you may try to:

  1. Unplug your microcontroller from your PC or power source, then plug it back in
  2. Press the Reset button on your microcontroller board
  3. Click Discover again in the Add Microcontroller Connection panel
