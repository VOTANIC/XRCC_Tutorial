3D Model

note

3D Model Blocks are available for 3D Model object type only.

BlockDescription
Play the animation of the model if the model contains animation, do nothing if otherwise.
Stop the animation if an animation is playing.

Set the material of the object.

  1. Material: Image of the material imported by thr user.

Set the material color of the object.

  1. Color: Color value specifying the material color.

Set the model to replace the current model.

  1. Model: 3D Model object added to the current scene.
  2. Material: Image asset in the project's asset library.
  3. Material Color: Color value specifying the material color.
